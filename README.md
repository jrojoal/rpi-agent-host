# rpi-agent-host

Automatització de publicació i gestió d’un agent per Raspberry Pi via SSH, amb integració DevOps i suport PowerShell.

## Descripció
Script PowerShell que permet:
- Crear repositoris GitHub automàticament
- Fer el `push` inicial
- Gestionar autenticació via GitHub CLI (`gh`)
